# Archived Legacy Code

This directory contains legacy code that is no longer used by the production Next.js application but is preserved for reference.

## Contents

- **Root-level JS files**: Standalone JavaScript files that were part of earlier versions
- **Legacy Games**: Old HTML-based games (moved to separate directory)
- **Test Scripts**: Old testing scripts
- **Solana Program**: Anchor/Solana smart contract code (separate from Next.js app)
- **Portfolio**: Old static HTML portfolio site

## Current Application

The current production application is a Next.js app located in:
- `/app` - Next.js app directory
- `/components` - React components
- `/lib` - Utility libraries
- `/public` - Static assets

All game functionality (including Hyperborea) is now integrated into the Next.js app at `/app/game`.
